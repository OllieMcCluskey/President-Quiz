# President-Quiz
**Vresion:1.0**

**Features**
* Questions about Presidents
* Answer verification
* images of presidents

**Tools used in this project**
* Images
* Labels
* Text boxes
* Buttons

**How to use the Source files**
* Go in to the source folder and download the aia file
* Go to [MIT App Inventor](https://ai2.appinventor.mit.edu/#4928864299057152)
* Press the import project button next to the New Project button
* Insert the aia file

**Diffrecnces from v1.0**
* The submit button will be disabled once the correct answer is guessed
* The next button is disabled until you answer the question no matter if its incorrect or correct
* Added a reset button which resets everything to what is was when you first opened the app
* Added a screen which shows up when all the answers were answered including text that tells you how many questions were answerd wrong and if all questions were answered correctly you will receive a text congradulating you
* Added recognition that will tell you to reanswer the question if you guessed the correct answer but uncapatalized and or has a space at the end of the word
